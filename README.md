# [Global Function Modeling and Minimization with SOS Bayesian Optimization](paper.pdf)
Lorenzo Shaikewitz (MIT) and Frederike Dumbgen (CMU)

This repo contains the RSS workshop paper and a minimal demo in Julia aligned with the toy example. To get started, open julia in this directory and instantiate the package:

```
julia --project
] instantiate
```

You can now use the jupyter notebook.
